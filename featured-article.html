JULY 23, 2022

My new journey as a bootcamp student.

As I progress through Scrimba's interactive courses, I'm gradually honing my HTML, CSS, and JavaScript skills, building a solid foundation for my career as a frontend developer. This site uses, Vite, PostCSS, grid and subgrid!

How I stay committed to learning

I like to think of myself as a lifelong learner. I used to spend hours and hours learning, then try to create simple projects using what I learned or work new techniques into existing projects.

For example, in this project I have experimented with the Space Toggle idea in CSS. What the Space Toggle does is enable a rudimentary type of if else statement in CSS. On this page for instance, I set the custom property controlling the uppercase of the first p to an empty space which invalidated the uppercase text-transform at runtime.

After this, I used a PostCSS plugin which enabled me to use the @custom-media rule so at medium screen sizes I can simply use the custom media query --medium-width.

That helped me further my CSS skills through applying new things, and I realized that posting my learning summaries was also helping others learn and stay motivated.

JULY 23, 2022

Grid autofit

On this website there is an auto-fit to display the post card so when the screen grows in size more columns are added automatically to add new posts instead of having to use @media or @container to figure out each individual place to put the post card. The individual post cards then span 1 column each!

JULY 23, 2022

Flexbox

For the header I briefly tried to use Grid. However, grid was not that great a fit for a 2d layout like that and so I chose to use flexbox with justify-content: space-between; to get them to go to either end of their space from the main grid which limited them to the content part of the grid.
